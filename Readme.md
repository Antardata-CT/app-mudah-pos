
# Mudah POS

Mudah POS merupakan aplikasi Point Of Sales penunjang kegiatan transaksional dan manajemen dari program I'm the boss Muda Berdaya Hebat dari **Kementrian Sosial Republik Indonesia**. Didalam aplikasi ini terdapat berbagai fitur penunjang dari fitur *Kasir* untuk proses transaksi, hingga manajemen stok.


## Fitur Aplikasi

- Menu Kasir
- Permintaan Barang dari Mitra ke Koordinator Lapangan
- Permintaan Barang dari Koordinator Lapangan ke Pusat
- Mutasi Barang ke masing-masing PIC di Lapangan
- Manajemen pengguna
- Manajemen Unit (Kendaraan)
- Grafik Penjualan
